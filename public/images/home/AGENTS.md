## Home Images

- Homepage-specific visual assets live here.
- Use descriptive kebab-case names such as `hero-portrait.webp`, `wordmark-zero-yue.webp`, and `work-artstation-design-system.webp`.
- Prefer final optimized web assets (`.webp`, `.avif`, `.png`) and keep source files outside `public`.
- When replacing placeholder paths, update `src/data/home.ts` first.

Current homepage assets:

- `home-portrait-main.png`: main character visual used by the homepage hero.
- `zero-yue-signature.png`: transparent handwritten signature layered around the hero title.
- `zero-yue-brand-mark.svg`: sidebar brand mark.
